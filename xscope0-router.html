xScope0 Router

An AI API proxy router with provider automation — one unified endpoint in front of many providers, with automatic failover, rotation and account-pool management.

v0.5.12 node >=18 license MIT JavaScript

#Overview

xScope0 Router sits between your client and a pool of AI providers. You point your app at a single local endpoint, and the router handles authentication, account rotation and failover behind the scenes. It ships with a dashboard for managing accounts and watching traffic.

Dashboard

http://localhost:20128/dashboard

API endpoint

http://localhost:20128/v1

Heads up This is a modified fork of 9router with custom branding, hardened TokenSaver paths and the donate / remote UI removed.

#Features

Core

  • Unified API proxy endpoint for multiple AI providers
  • Automatic provider failover and rotation
  • Account-pool management with bulk operations
  • SQLite-backed persistent storage
  • System-tray integration (macOS / Linux / Windows)

Provider automation

  • Mimo Code Free + OpenCode Free proxy automation
  • Proxy-pool selection with auto-rotation on 5 / 10 / 15-minute intervals
  • Automatic rotation on rate-limit or provider errors (incl. Mimo 400 / 441)
  • Provider-wide Error → inactive policy for account providers
  • Kiro temporary-suspension handling

Account management

  • Bulk-delete selected, inactive or deactivated accounts
  • Bulk API-key import (appends; does not replace same-name accounts)
  • Proxy-pool force-delete unbinds accounts first

#Install

Install globally and run the binary:

npm install -g xscope0-modifed-router
9router

Or run it directly without installing:

npx xscope0-modifed-router
Tip The package exposes both 9router and xscope0-router binaries for compatibility.
