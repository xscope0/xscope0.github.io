MuffinStoreJailed

On-device App Store client for iOS app downgrades. No jailbreak required.

Swift ios 14.0+ deprecated · PoC AGPL-3.0
Status This is a proof of concept, not a final product. Use TrollStore MuffinStore instead if you can.

#Overview

MuffinStoreJailed is a fork of mineek/MuffinStoreJailed-Public. It runs entirely on-device and lets you roll an installed app back to an older release without a jailbreak.

#What it does

Downgrade any iOS app to a previous version. It exploits Apple's App Store download API to fetch older IPA files signed with your Apple ID — and works without a jailbreak.

#How to use

  1. Uninstall the target app The app must be removed before downgrading. To keep your data, offload it first (Settings → General → iPhone Storage → Offload App), then uninstall.
  2. Request a 2FA code Before logging in, open Settings and request a 2FA code. On iOS 18+ the Settings option may not appear — leave the code field empty in the app and a popup will ask you to approve the login; copy the code from there. If login fails, force-close the app and retry.
  3. Sign in & search Sign in with your Apple ID, then search for the app you want to downgrade.
  4. Select the target version Pick the version you want from the list.
  5. Wait for the install popup There is no progress bar. Stay on the screen until an installation popup appears — time depends on app size.
  6. Install & finalize Press Install, wait ~5 seconds, then return to SpringBoard. The downgrade finalizes automatically.
No progress bar Downloads give no visual feedback. Do not leave the screen until the install popup shows.

#Known issues

  • No progress indicator during download
  • UI is rough — this is a PoC
  • 2FA flow is inconsistent across iOS versions
  • Some apps may not have older versions available on Apple's CDN

#Build

git clone https://github.com/xscope0/MuffinStoreJailed.git
cd MuffinStoreJailed
make

Or open MuffinStoreJailed.xcodeproj in Xcode.

Requirements: iOS 14.0+, Xcode 14+, and a sideload method (TrollStore / AltStore / Sideloadly).

#Install

Grab a pre-built IPA from Releases, then sideload it:

MethodSteps
TrollStoreDownload .ipa, open with TrollStore
AltStoreSideload via AltStore. Re-sign every 7 days (free ID)
SideloadlySideload via Sideloadly

#Disclaimer

Experimental tool — use at your own risk. Apple ID credentials are used locally; nothing is sent to third-party servers. Apple may detect unusual API usage. No warranty, no guarantee. Downgraded apps may have compatibility issues with newer iOS versions.
